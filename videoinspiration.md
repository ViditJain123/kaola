# Video Inspiration

A living library of short-form video scripts that work in the market.
This file is the single reference for "what good looks like" — hooks,
pacing, structure, tone.

Two ways things land here:

1. **Manually** — paste a script you like below, under a `##` heading.
2. **Automatically** — run `/content-machine:add-inspiration <tiktok-url>`
   and the plugin transcribes + appends a new entry.

The `generate-script` skill reads this whole file and learns patterns
from it (hooks, cadence, angles) — it never copies lines verbatim.
