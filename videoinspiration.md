# Video Inspiration

The single reference for "what good looks like" — hooks, pacing,
structure, tone. The `generate-script` skill reads this whole file and
learns patterns from it (hooks, cadence, angles) — it never copies
lines verbatim.

The three scripts below are the **gold-standard koala-voice scripts**
that every new generation should resemble. They share a common DNA:

- **Hook is a question + "explained by koalas"** (lowercase, plural,
  question-mark, no period before "explained").
- **Tone is conversational philosopher**, not broken-syntax raccoon.
  Lowercase throughout. Em-dashes for pivots. Period-heavy cadence
  for emphasis ("cheap. fast. infinite.").
- **Direct address** — uses "you" liberally alongside third-person
  "koala". The viewer is in the scene, not being lectured.
- **Same core insight every time** — the brain isn't addicted to the
  phone, it's addicted to dopamine. Willpower can't beat chemistry.
  Redirect, don't fight.
- **Same fix every time** — build one off-screen habit, then *film
  yourself doing it*. Timelapse is the better dopamine source.
- **Same CTA every time** — "koala doesn't gatekeep, koala use
  lockin club to track his productivity using timelapses."

## What to copy and what to invent

The above three scripts should anchor every new generation. New
scripts should:

- **Copy structurally** — the 5-beat arc (hook → toxic cycle →
  reframe → fix → CTA), the lowercase tone, the em-dash pivots, the
  period-stacked cadence.
- **Copy phrase-shape, not phrases** — "explained by koalas." is the
  signature. "same loop. better source." is the *kind* of closer to
  invent — three or four words, two sentences, one pivot. Don't copy
  it verbatim.
- **Vary the hook angle** — stat-driven (60 days), conspiratorial
  whisper (pssst), contrarian question (why does willpower never
  work). The *shape* is constant (question + "explained by koalas")
  but the *angle* should rotate.
- **Vary the off-screen habits** — running, reading, meditation,
  fishing are the established set. Add new concrete habits if the
  angle calls for it (woodworking, climbing, drawing) — never
  abstract ones ("being present," "self-care").
- **Always end on the same CTA** — "koala doesn't gatekeep, koala
  use lockin club to track his productivity using timelapses." This
  is fixed. Vary the line *before* the CTA, not the CTA itself.
